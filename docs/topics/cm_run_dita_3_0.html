Running DITA OT x.x.x

The SWCMS saves the DITA and markdown files in the <DITA_OT_DIR>/samples/topics directory. The bookmap files are stored one level above: <DITA_OT_DIR>/samples.

Run the following command to create a PDF file:

 ../bin/dita --input=cm_bookmap.ditamap --format=pdf --output=out

Run the following command to create a HTML file:

 ../bin/dita --input=topic_map_cms.ditamap --format=xhtml --output=out

Note: This example assumes that you have already created a bookmap.

Run the following file to create an HTML file:

../bin/dita --input=topicmap_ditacms.ditamap --format=xhtml --output=out1

The following are possible formats to use with DITA OT x.x.x:

  • dita

  • eclipsehelp

  • html5

  • htmlhelp

  • markdown, markdown_gitbook, and markdown_github

  • pdf

  • xhtml

Refer to External for further details.
